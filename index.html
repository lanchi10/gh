מדריך למתחילים ב-GitHub

1. התחברות ל-GitHub

היכנס לחשבון GitHub שלך או צור חשבון חדש אם עדיין אין לך.

טיפ למובייל: השתמש באפליקציית GitHub לגישה נוחה יותר.
למד עוד על יצירת חשבון GitHub

2. יצירת מאגר חדש

לחץ על סימן ה-+ בפינה העליונה ובחר 'New repository'. תן שם למאגר ולחץ 'Create repository'.

טיפ למובייל: גלול למטה כדי למצוא את כפתור 'Create repository'.
מדריך מפורט ליצירת מאגר חדש

3. העלאת הקובץ הראשון

בחר 'uploading an existing file', בחר את קובץ ה-index שלך מהטלפון, הוסף תיאור ולחץ 'Commit changes'.

טיפ למובייל: אם אתה מעלה קובץ מהטלפון, ודא שהוא נמצא בזיכרון הפנימי ולא בענן.
הסבר מפורט על העלאת קבצים ל-GitHub

4. מציאת הלינק לאינטרנט

חזור לעמוד הראשי של המאגר, לחץ על הקובץ, אז על 'Raw', והעתק את כתובת ה-URL מהדפדפן.

טיפ למובייל: לחץ לחיצה ארוכה על שורת הכתובת כדי להעתיק את ה-URL.
מידע נוסף על צפייה בקבצים ב-GitHub

5. שימוש בסיסי ב-GitHub

כעת תוכל לערוך את הקובץ ישירות באתר, להוסיף קבצים נוספים, ולעקוב אחר השינויים.

טיפ למובייל: השתמש ב-'Edit file' כדי לבצע שינויים קטנים ישירות מהטלפון.
